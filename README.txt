Mantiq is a workflow automation tool. It allows you to build the backend you need in your WordPress website with LEGO-like building blocks instead of writing code.

== Description ==

### Mantiq WordPress Workflow Automation Plugin

Mantiq allows you to craft customized workflows, without leaving your WordPress dashboard.

== Frequently Asked Questions ==

**Is Mantiq a self-hosted plugin?**

Yes, you have control over all data!

**How many workflows can I add?**

There is no feature limit, but it depends on your server capacity.

**What are the minimum requirements to run Mantiq?**

You'll need WordPress 5.0+, PHP 7.0+, and MySQL 5.7+

**Can developers extend Mantiq?**

Absolutely! For more details, please visit: [wpmantiq.com](https://wpmantiq.com/docs/?utm_source=wp-org&utm_medium=description&utm_campaign=mantiq).
